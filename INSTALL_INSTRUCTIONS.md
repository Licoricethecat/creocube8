# Install Instructions
The modpack is on 1.21.1 neoforge 21.1.234, but the pack files will automatically set the neoforge and minecraft versions in the launcher you're using.

The [releases](https://github.com/Licoricethecat/creocube8/releases) page provides modpack installs for Prism, Modrinth, and Curseforge so you can choose any of these installers as you like. My top recommendation is Prism, as it can download mods and resourcepacks available on both curseforge and modrinth and is easily customizable, but I also recommend Modrinth as a close second as it's clean and easy and doesn't install bloatware unlike curseforge.

## Prism (Recommended)
I highly recommend Prism if you've never played modded before, as it's really easy to import modpacks of all types and create your own modded instances how you'd like.
1. Install Prism (obviously lol) (make sure to download it from https://prismlauncher.org)

![prism step 1](https://github.com/Licoricethecat/creocube8/blob/main/images/install-instructions-images/prism%20step%201.png "Download from prismlauncher.org")

2. Click on "Add Instance" in the top left

![prism step 2](https://github.com/Licoricethecat/creocube8/blob/main/images/install-instructions-images/prism%20step%202.png)

3. Click on "Import" on the left side.

![prism step 3](https://github.com/Licoricethecat/creocube8/blob/main/images/install-instructions-images/prism%20step%203.png)

Then, browse for the ***Prism*** version of the modpack provided in [releases](https://github.com/Licoricethecat/creocube8/releases)

![prism step 3.5](https://github.com/Licoricethecat/creocube8/blob/main/images/install-instructions-images/prism%20step%203.5.png)

Then, hit "OK" in the bottom right to install the modpack!

![prism step 3.75](https://github.com/Licoricethecat/creocube8/blob/main/images/install-instructions-images/prism%20step%203.75.png)

4. Now, your instance of CC8 should appear on the main page! On the right, you can click "Launch" to start the game or "Edit" if you want to install your own resourcepacks, shaderpacks, or other client-side mods. (If you install more mods, make sure they're client-side only as mods not included in the pack install that require it to be on both client and server WILL NOT WORK.)

![prism step 4](https://github.com/Licoricethecat/creocube8/blob/main/images/install-instructions-images/prism%20step%204.png)

![prism step 4.5](https://github.com/Licoricethecat/creocube8/blob/main/images/install-instructions-images/prism%20step%204.5.png)

## Curseforge
Curseforge is my least recommended option (way too much bloatware), so I'd say to use it only if both Prism and Modrinth don't work. Basically all minecraft mods nowadays are posted on Curseforge though, so I do recommend getting familiar with Curseforge's mod pages.

https://www.curseforge.com/download/app
